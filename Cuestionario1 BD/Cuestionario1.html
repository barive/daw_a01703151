Preguntas

  • ¿Qué es entonces una base de datos - DB?
    Es un contenedor que permite almacenar la información de forma ordenada con diferentes propósitos y usos [1]. Es una colección de archivos interrelacionados, los datos pertenecen a un mismo contexto. Ayuda a una eficiente recuperación, inserción y eliminación de datos y los organiza en forma de tablas, esquemas, reportes, etc. [2]
  • ¿En qué casos es conveniente usar bases de datos?
    Cuando se maneja una gran cantidad de información o información muy pesada, dicha información está interrelacionada y además es accesada por muchos usuarios o aplicaciones.
  • ¿Qué es un sistema de gestión de base de datos - DBMS?
    Es el software que se utiliza para manejar las bases de datos [2]. Gestiona grandes bloques de información y sus estructuras.
  • Enlista y explica las funciones/responsabilidades que tienen los DBMS.
    • Abstacción de los datos: ahorrar detalles del almacenamiento físico de los datos.
    • Consistencia: vigilar que los datos redundantes se actualicen simultáneamente.
    • Seguridad: manejar un sistema de permisos de usuarios, evitar que personas ajenas manipulen los datos.
    • Integridad: proteger los datos ante cualquier tipo de fallo.
    • Respaldo: realizar copias de respaldo de los datos.
    • Control de la concurrencia: manejar el acceso simultáneo de muchos usuarios.
    • Manejo de transacciones: modificaciones de los datos.
    [3]
  • Ejemplifica y justifica en qué proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
    El proyecto que mi equipo hizo en Fundamentos de Diseño Interactivo. Se trataba de una página dirigida a estudiantes de preparatoria para aprender química. Hubiera sido interesante que la página guardara los datos de cada persona, como: nombre, lecciones revisadas y calificación de los quizzes.
