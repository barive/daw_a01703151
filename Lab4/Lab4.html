Preguntas

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    Los dos se asemejan en el aspecto de que ambos son lenguajes orientados a objetos; también en que ambos se utilizan para desarrollar tanto en back-end como el front-end. Se diferencias principalmente en que Java es un lenguaje compilado mientras que JavaScripts es un lenguaje interpretado (script). Otra gran diferencia es que Java es un lenguaje tipado, o sea, que los tipos de variables deben especificarse; por otro lado, JavaScript es no tipado. [1]
  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    • getFullYear()

    • push()
    • shift()
    • unshift()

  • ¿Cómo se declara una variable con alcance local dentro de una función?
    Se declara igual que cualquier otra variable, sólo que debe de estar a fuerzas dentro de la función. Cuando se hace esto la variable solamente se puede acceder desde dentro de la función, es decir, tiene alcance local. Estas variables locales se borran al terminar la función. [4]
  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    Implica que las variables globales se eliminan hasta que se cierra la ventana del navegador, pero permanecen disponibles para nuevas páginas abiertas en la misma ventana. [4]
  • ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?
    Se puede utilizar el método match(). Este método busca una combinación en un string y regresa esas coincidencias como un objeto array. Se puede usar en una página web si se busca algo en específico que tiende a repetirse. [5]
